Upload your ROIs

You can upload a BED file containing one or multiple rois. The file must contain the following elements separated by tab:

  • Description:
  • 1. Column: Chromosome name written as 'chr#'
  • 2. Column: Start position index
  • 3. Column: End position index
  • 4. Column: ROI-name
  • 5. Column: 0 (Placeholder for scores)
  • 6. Column: Strand
  • 7. Column: 0 for searching only within the framesize, 1 for searching the whole region

Please note, that the file must not end with an empty line or whitespace
