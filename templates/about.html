About CrisprApp

This website helps you to find and select your CRISPR targets for one or multiple Regions of Interest (ROI)

CrisprApp uses the following scoring methods:

The Doench Score calculates the on-target activity using the method proposed by Doench et al, 2014. This scoring algorithm analyses the one- and two-base features of the gRNA, as well as the GC content, and uses an experimentally determined predictive model to score the expected activity level of the CRISPR target. Scores are between 0 and 1, with a higher score denoting higher expected activity. Note that ambiguous bases wonâ€™t contribute to this score. Doench 2014

The MIT Specificity score summarizes all off-targets into a single number from 0-100. The higher the number, the fewer off-target effects are expected. We recommend targets with an MIT specificity > 50

The GC content is usually calculated as a percentage value and sometimes called G+C ratio or GC-ratio. GC-content percentage is calculated as Count(G + C)/Count(A + T + G + C) * 100%. We recommend a GC content score between 40 to 80%
